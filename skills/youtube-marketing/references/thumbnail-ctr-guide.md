# Thumbnail CTR Guide

## What Drives CTR (Data-Backed)

| Element | CTR Impact | Source |
|---------|-----------|--------|
| Face in thumbnail | +38% | vidIQ, 5M channel dataset |
| Dark background | Higher in tech/tutorial | SERP pattern analysis |
| High contrast text | Readable at small size | A/B testing data |
| Result shown (not process) | Higher than "building" state | Focus Digital, 2025 |
| 2–3 words max | Better than 5+ words | vidIQ analysis |

## The Winning Formula (Tutorial / Tech Niche)

```
Background: DARK (black or very dark gray — not white, not grey)
Subject:    The OUTPUT in its final, revealed state — NOT the Elementor editor
Text:       2–3 words max, bold, high contrast (white or brand accent)
Concept:    Viewer instantly sees: "I want that result"
```

**What NOT to show**: The Elementor backend while building. Nobody clicks on a screenshot of panel settings. Show the finished website in action.

## 3 Proven Thumbnail Archetypes

### Type A — Result Showcase
- Full-width finished UI on dark background
- 2–3 word overlay stating the outcome
- Best for: widget demos, design tutorials
- Example: Dark background + the animated side menu in open state + "HOVER MENU"

### Type B — Before/After Split
- Left: plain/boring version | Right: polished version
- Visual contrast = instant curiosity
- Best for: comparison, transformation, "fix this" tutorials
- Example: Default Elementor nav | The fancy hover sidebar

### Type C — Text-Heavy (Bold Statement)
- Large typography dominates
- For when the words ARE the hook
- Best for: controversial or curiosity-gap titles
- Example: "NOT A POPUP" in huge text

## Title-Thumbnail Synergy Check

Your title and thumbnail must not say the same thing — they should complement each other:

| Title says | Thumbnail shows |
|-----------|----------------|
| "No Code Needed" | The polished result (implies code-level quality) |
| "Smooth Hover Menu" | Menu in motion / revealed state |
| "NOT Off-Canvas Popup" | The alternative design they'd prefer |

If both say the same thing, you're wasting one signal. Thumbnail = visual promise, Title = keyword search + hook.

## Thumbnail Text Rules

- Max 3 words (ideally 2)
- Use CAPS for 1–2 power words only
- Don't put the full video title on the thumbnail
- Readable at 120px × 68px (YouTube search size) — test by zooming out

## A/B Testing

YouTube Studio shows title/thumbnail A/B results after ~1,000 impressions.
- Run tests for minimum 7 days before deciding
- Change one variable at a time (thumbnail OR title, not both)
- The winning variant becomes permanent — update within 2 weeks
