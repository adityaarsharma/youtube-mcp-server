# YouTube SEO Playbook

## Title Formula

```
[Primary keyword — first 40 chars] | [Differentiator] | [Brand/Year]
```

**Non-negotiables:**
- Primary keyword in first 40 characters (highest indexing weight)
- Under 100 characters total (YouTube hard limit)
- At least one differentiator word: FREE / Without [X] / No Code / Smooth / Animated
- Year (2026) in at least 1 of your 3 title variants
- Never start with "How to" alone — too generic, too low CTR

**High-CTR words proven in tutorial niche (SERP data, 2026):**
- INCREDIBLE / SMOOTH / BEAUTIFUL / ANIMATED → emotional
- WITHOUT [thing they hate] / NO CODE / NO CSS / FREE → pain-point relief
- COMPLETE / FULL TUTORIAL / STEP BY STEP → thoroughness promise
- [Year] → freshness signal
- Portfolio / Agency / Beginner → audience callout

## Description Architecture

**First 150 chars** (shown before "Show more") = highest keyword weight:
- Must contain primary keyword naturally
- Must name the product/tool
- Do not start with widget URL — put URL on line 1, start text immediately after

**"WHAT YOU'LL LEARN" section** = critical for indexing:
- Each bullet is an additional keyword signal
- Write bullets as "How to [action] + [outcome]" — both searchable and motivating
- 5–7 bullets max

**Links section**: Fix it once, use it in every video. Inconsistent links = missed traffic.

## Tag Strategy (20 tags, ordered by priority)

1. Exact primary keyword
2. Exact secondary keyword
3. Product + feature (e.g., "wdesignkit hover menu")
4. Broader category (e.g., "elementor navigation tutorial")
5. Use case (e.g., "portfolio navigation wordpress")
6. Competitor method (e.g., "off canvas menu elementor") — captures comparison searches
7. Product brand (e.g., "wdesignkit", "the plus addons")
8. Year variant
9. Platform broad (e.g., "elementor", "elementor pro")
10. WordPress general

**Hard limits**: No tag over 30 chars. Total under 500 chars. No keyword stuffed (same word 8+ times).

## SERP Gap Analysis (Run Before Every Video)

Search the primary keyword with `serp_youtube_organic_live_advanced`. Look for:

| Signal | What to Do |
|--------|-----------|
| Top video >2 years old | Freshness gap — make this video now |
| Top video requires CSS/code | No-code gap — your widget version wins |
| Top video requires Pro plugin | Free tier gap — target with "without Pro" framing |
| No video covers full use case | Completeness gap — go comprehensive |
| Top video <100K views | Moderate opportunity |
| Top video >100K views | Proven demand — strong opportunity |

## Chapters (Timestamps)

Chapters are a ranking signal — they create additional keyword-indexed anchors in your video.
- Minimum 3 timestamps
- Must start at `0:00`
- Each chapter title = additional keyword opportunity
- Format: `0:00 Introduction` (no brackets required)
