# YouTube Analytics Benchmarks

Reference data for diagnosing channel health. All benchmarks are for **tutorial / how-to / software channels** unless noted.

## CTR Benchmarks (Focus Digital, Dec 2025)

| Grade | CTR | Diagnosis |
|-------|-----|-----------|
| A | >8% | Excellent — thumbnail + title working well |
| B | 5–8% | Good — room to improve thumbnail |
| C | 3–5% | Average — test new title or thumbnail |
| D | <3% | Problem — thumbnail or title is not working |

**Impression source matters**: Search impressions typically have lower CTR (1–3%) than Browse (4–8%). Compare by source, not just overall.

## Average View Duration Benchmarks

| Grade | % Retained | Diagnosis |
|-------|-----------|-----------|
| A | >50% | Excellent retention |
| B | 35–50% | Healthy — algorithm will recommend |
| C | 20–35% | Average — hook or content pacing issue |
| D | <20% | Problem — hook is losing people fast |

**Absolute duration**: For tutorial niche, 7–15 min videos are optimal for search ranking (Briggsby, 2024). Under 7 min ranks less competitively for difficult keywords.

## Subscriber Conversion Benchmarks

| Grade | Sub/View Ratio | Diagnosis |
|-------|---------------|-----------|
| A | >1% | Excellent — strong value + CTA |
| B | 0.3–1% | Healthy |
| C | 0.1–0.3% | Weak CTA or audience mismatch |
| D | <0.1% | Broken — CTA missing or wrong audience |

**Fix for D**: Add CTA at 25% mark (not end), use `?sub_confirmation=1` in subscribe link, check if audience matches content.

## Traffic Source Health

| Source | Healthy | Warning | Problem |
|--------|---------|---------|---------|
| YouTube Search | 30–50% | 20–30% | <20% |
| Browse / Home | 20–35% | 10–20% | <10% |
| Suggested Videos | 15–30% | 5–15% | <5% |
| External | 5–15% | — | >40% |

| Metric | Good | Average | Weak |

| Comment/view ratio | >0.5% | 0.1–0.5% | <0.1% |
