# Monetization Guide

## The Product-Company YouTube Revenue Model

Standard YouTube advice is built for ad revenue. If you're a product company, your model is different:

```
Ad revenue model:  Views → CPM → $0.001 per view
Product model:     Views → Free download → Pro upgrade → $49–200 LTV
```

One plugin sale = 50,000 ad views. Optimize accordingly.

## Revenue Priority Order

| Priority | Stream | Unit LTV | Effort to Start |
|----------|--------|----------|----------------|
| 1st | Plugin/product sales | $49–200 | Zero (already in description) |
| 2nd | Email list capture | $30–120 lifetime | Low (add lead magnet) |
| 3rd | Affiliate (hosting, tools) | $15–150/referral | Low (add links) |
| 4th | Ad revenue (AdSense) | $0.001–0.004/view | None (passive) |
| 5th | Sponsorships | $200–800/video | High (relationships) |
| 6th | Course/membership | $49–99/student | Very high |

## Plugin Sales Optimization

The funnel:
```
Video → Description link (top 5 lines) → Widget landing page → Free download → Pro upgrade
```

Every step must be present and working:
- Link in first 5 lines of description (not buried below "Show more")
- Live demo on landing page (doubles conversion — ConversionXL, 2024)
- Free tier available (reduces friction, converts to paid over time)
- Discount code in description (reduces purchase hesitation)
- CTA mentioned at 25% mark in video (not just end)

## Affiliate Partnerships (No Product Conflict)

Best WordPress-niche affiliates:
- Hosting: WP Engine ($200), Cloudways ($50–100), Hostinger ($60–80) per referral
- Page builders/tools: Rank Math Pro ($15–40), WP Rocket ($15–30)
- Design: Envato Elements ($10–25/referral)

Rules: Only promote what you actually use. One bad sponsored integration loses 3 months of trust.

## CPM Seasonality

| Quarter | CPM vs avg | Strategy |
|---------|-----------|---------|
| Q4 (Oct–Dec) | +40–60% | Push product CTAs, run sale promos |
| Q1 (Jan) | Lowest | Volume play — upload more, push free tier |
| Q2–Q3 | Baseline | Standard mix |

Black Friday week = highest plugin purchase intent in the year. Plan a sale announcement video.
