# Channel Template — WordPress Specialist

## When to Use
Use this template when:
- Channel covers WordPress ecosystem (not a specific product)
- Revenue = affiliate (hosting, plugins, themes) + AdSense + sponsorships
- Audience = WordPress site owners, developers, designers

## Content Strategy

**Evergreen keywords** (consistent volume year-round):
- "How to install [plugin]"
- "[Plugin] tutorial"
- "[Page builder] vs [Page builder]"
- "How to speed up WordPress"
- "WordPress backup tutorial"

**Seasonal spikes** (Semrush 2025):
- Jan–Mar: "Best WordPress plugins 2026", "WordPress SEO guide"
- Sep–Nov: "WordPress tutorial", learning intent picks back up

**High-affiliate-value content**:
- Hosting comparisons → highest commission
- "Best [type] plugin" roundups → multiple affiliate links
- "I switched from X to Y" → authentic narrative + affiliate

## Thumbnail Style

WordPress tutorial audience responds to:
- Before/after website screenshots
- "Stop doing X" style thumbnails (fear of missing out)
- Clean, readable text at thumbnail size
- Familiar WordPress dashboard UI shown = instant recognition
