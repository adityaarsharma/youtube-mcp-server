# Channel Template — SaaS Channel

## When to Use
Use this template when:
- Product is a SaaS (monthly/annual subscription)
- Goal = free trial signups → paid conversions
- Audience = business users evaluating tools

## Channel Constants

```
Product name: [SaaS product name]
Free trial URL: [URL]
Pricing page: [URL]
Docs: [URL]
Help center: [URL]
Community: [Slack/Discord URL]
Promo code: [CODE] ([N]% off first month/year)
Subscribe URL: https://www.youtube.com/c/[handle]/?sub_confirmation=1
```

## Content Mix

| Type | % | Purpose |
|------|---|---------|
| Use case tutorials | 40% | "How to [job to be done] with [Product]" |
| Integration demos | 25% | "Connect [Product] with [Popular Tool]" |
| Feature spotlights | 20% | New feature → existing user retention |
| Competitor comparisons | 15% | "[Product] vs [Competitor]" — high buyer intent |

## CTA Strategy

- Primary CTA: "Start your free trial" — no credit card required is the magic phrase
- CTA at 25% mark: "Link in description to try this yourself"
- End screen: "Try [Product] free" card + subscribe
- Never pitch in first 30s — let the demo sell

## Keyword Strategy

SaaS YouTube keywords differ from tutorial channels:
- High buyer intent: "[Product name] tutorial", "[Product name] review", "[Product] vs [Competitor]"
- Job-to-be-done: "how to [specific task] [industry]"
- Integration: "[Product] + [popular tool]" (Zapier, Slack, etc.)
- Problem-first: "how to solve [specific pain point]"

## Revenue Funnel

```
YouTube View
    ↓
Free Trial Signup (via description link)
    ↓
Onboarding Sequence (email)
    ↓
Power User Behavior (product activation)
    ↓
Paid Conversion ($X/month)
    ↓
Annual Plan Upgrade
```

## Publish Strategy

- Feature release videos: publish on same day as product release (highest intent)
- Tutorial videos: publish Mon–Thu, 9–11 AM in target audience timezone (business hours)
- Shorts: product tips — "Did you know [Product] can..." format
