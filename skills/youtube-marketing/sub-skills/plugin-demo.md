# /youtube plugin-demo

## Why Plugin Demos Are Different
- Viewer intent: "Can this plugin solve my problem?" — not just "how to do X"
- **ConversionXL, 2024**: Plugin demo videos convert at 2.8% avg, 6%+ for widget demos
- **Semrush, 2025**: "How to [plugin feature]" queries show YouTube videos in top 3 Google results 67% of the time
- Goal: views + docs traffic + trial/purchase — all three, not just views

## Title Formula for Plugin Demos

```
[Build/Add/Create] + [OUTCOME the user wants] + [in Platform] | [Differentiator] | [Product + Year]
```

**Examples:**
- "Build Sticky Header in Elementor | Scroll Effects + Transparent | The Plus Addons 2026"
- "Add Mega Menu in WordPress WITHOUT Elementor Pro | The Plus Addons 2026"
- "Create Team Section in Elementor | Hover Animations + Profile Cards | WDesignKit 2026"

**What makes plugin demo titles work:**
- Lead with the OUTPUT (what the user gets), not the widget name
- "WITHOUT [competing tool]" is the strongest differentiator when competitors require paid tools
- Year signals freshness and ranks above aged tutorials

## Demo Video Structure (13–18 minutes optimal)

```
0:00 – 0:40  HOOK: Show the finished result in action, then say what we're building
0:40 – 1:30  WHAT YOU'LL LEARN: 6-8 outcomes, spoken
1:30 – 2:30  INSTALL + ACTIVATE: Fast, no dwelling — viewer already has the plugin or is curious
2:30 – 3:00  WIDGET ACTIVATION: Where to find and enable
3:00 – 5:00  SCRATCH BUILD: Build the widget from zero (important — shows the learning curve is low)
5:00 – 6:30  IMPORT DEMO: Show the one-click template import
              👆 This is the highest-value moment — show it clearly
6:30 – 10:00 KEY FEATURES WALKTHROUGH: Feature by feature, result-first (show result → explain control)
10:00 – 11:30 ADVANCED OPTIONS: Sticky, animations, responsive — for power users
11:30 – 12:30 FINAL RESULT: Full demo of the finished output
12:30 – 13:00 CTA: "Try it free → link in description" + Subscribe
```

## CTA Strategy for Plugin Videos

**Three CTAs, placed strategically:**
1. **At 25% mark** (soft): "Subscribe if this is helpful — new Elementor tutorials every week"
2. **At import demo moment**: "The link to get this widget is in the description"
3. **End screen**: "Try the free version → link in description" + "Watch this next → [related video]"

**Description CTA order (critical):**
1. Widget page link (first — highest intent)
2. Live demo link (second — lets them see result before buying)
3. Docs link (third — support intent)
4. Free download, then Pro pricing

## SEO Specific to Plugin Demos

**Title patterns that rank for product demos:**
- "How to [Feature] in Elementor" → captures learning intent
- "[Feature] in Elementor WITHOUT [Competitor/Pro]" → captures comparison intent
- "Best [Feature] Plugin for Elementor" → captures decision intent (high buyer intent)

**Description WHAT YOU'LL LEARN for plugin demos:**
Always include these specific bullets:
- ✅ How to install and activate [Plugin Name]
- ✅ How to enable the [Widget Name] widget
- ✅ How to import the pre-built demo template
- [Feature-specific bullets]
- ✅ How to customise for [specific use case — agency/portfolio/etc.]

**Tags for plugin demos — always include:**
- "[widget name] elementor"
- "[plugin name] elementor"
- "[plugin name]" (brand alone)
- "elementor [widget type] widget"
- "wordpress [feature] plugin"
