# /youtube strategy

## Research Foundation
- **Demand Curve, 2025**: Tutorial channels follow power law — 2 broad "breakout" attempts per month, rest can be niche
- **Semrush, 2025**: WordPress plugin tutorials spike Jan–Mar and Sep–Nov; avoid holiday weeks
- **vidIQ 5M-channel data**: 2–3x/week grows subs 4x faster than 1x/week
- **ConversionXL, 2024**: Plugin demo videos convert at 2.8% avg; live demo on landing page pushes to 6%+
- **YouTube Internal, 2024**: "Expert" channels (consistent niche, consistent format) get 30% more Suggested placements
- **Backlinko 2024**: Channels posting >52 videos/year rank in YouTube search 2.3x more often

## POSIMYTH Channel Framework

### The Product-Company YouTube Model (Different From Ad-Revenue)

Most YouTube strategy is built for ad revenue. POSIMYTH's model is different:

```
Standard creator:  Views → Impressions → Ad Revenue
POSIMYTH model:    Views → Landing Page → Free Download → Pro Upgrade
```

**This means:**
- CTR matters for volume, but **conversion rate** (view → plugin install) matters more
- Long-form content (8–15 min) performs better than Shorts for plugin purchases
- Audience = WordPress developers + designers making purchasing decisions
- Every video is a sales asset, not just content

### Channel Positioning Statement

```
POSIMYTH is the #1 source for no-code, no-CSS Elementor tutorials using 
The Plus Addons, WDesignKit, and NexterWP — for web designers, agencies, 
and developers who want premium results without premium complexity.
```

**Owned angles** (no competitor covers these together):
1. Widget-powered results with zero code
2. Full agency/portfolio use cases (not just beginner demos)
3. All POSIMYTH products cross-linked in every video (internal audience migration)

## Content Pillars

### Pillar 1 — Broad Elementor Tutorials (40% of content)
**Purpose**: Discoverability. Capture search traffic from users who don't know POSIMYTH yet.
**Format**: 8–15 min full build tutorial
**Keywords**: "elementor [feature]", "how to [build X] in wordpress"
**CTA**: Mention POSIMYTH product as the tool used — not forced, just natural

### Pillar 2 — Widget/Feature Demos (35% of content)
**Purpose**: Conversion. Viewers actively evaluating the plugin.
**Format**: 6–12 min widget walkthrough with live demo
**Keywords**: "[widget name] elementor", "[feature] elementor no code"
**CTA**: Direct link to widget landing page + docs in description

### Pillar 3 — Comparison and Problem-Solving (15% of content)
**Purpose**: Win searches with "without X / no X" framing. Captures Pro users considering alternatives.
**Format**: 5–10 min focused answer
**Keywords**: "[feature] without elementor pro", "[thing] free wordpress"
**CTA**: POSIMYTH product as the free/easier alternative

### Pillar 4 — Shorts and Quick Tips (10% of content)
**Purpose**: Shorts feed impressions → channel awareness → subscriber growth
**Format**: 30–60s vertical
**Rule**: Post within 7 days of long-form video (freshness decay cliff at 28–30 days)

## Upload Cadence

| Goal | Cadence | Ratio |
|------|---------|-------|
| Aggressive growth | 3x/week + 3 Shorts | 2:1 broad:widget |
| Sustainable growth | 2x/week + 2 Shorts | 2:1 broad:widget |
| Maintenance | 1x/week + 1 Short | 2:1 broad:widget |

**Minimum for algorithm favor**: 2 videos/week (vidIQ: 4x subscriber growth vs 1x/week)

**Best publish days**: Monday, Tuesday, Wednesday, Thursday  
**Best publish time**: 9:00–11:00 PM IST (3:30–5:30 PM UTC)  
**Gap between uploads**: Minimum 3 days (algorithm momentum window)

## Target Audience Segments

| Segment | Size | Intent | Best Content |
|---------|------|--------|--------------|
| Freelance web designers | Largest | Learn + do | Full build tutorials |
| Agency owners | High value | Evaluate tools | Widget demos + "agency" callout |
| Elementor beginners | Large | Learn basics | Pillar 1 broad content |
| Pro users considering alternatives | High value | Compare | Pillar 3 "without Pro" |
| WordPress developers | Technical | Reference | Advanced feature demos |

**Underserved by competitors**: Agency-focused tutorials with professional outcomes (not beginner "Hello World" builds)

## 30/60/90-Day Milestone Plan

### Days 1–30: Foundation
- [ ] Audit all existing videos — apply batch SEO to bottom 20 by CTR
- [ ] Fix subscriber conversion: add end screen CTA at 30s before end on all top 20 videos
- [ ] Establish publish schedule: 2x/week, same days each week
- [ ] Produce 8 videos: 5 broad + 3 widget demos
- [ ] Target: 5% CTR avg on new videos, 1 sub per 50 views

### Days 31–60: Acceleration
- [ ] Identify top 2 videos from Month 1 → make follow-up videos on same topic
- [ ] Launch Shorts strategy: clip 1 Short from each long-form video
- [ ] Build one "mega" video: 15+ min complete project (e.g., "Build an Agency Portfolio Site from Scratch")
- [ ] Target: 50+ subs/day from content alone (no promotions)

### Days 61–90: Compound
- [ ] Identify first "breakout" video candidate (10K+ views trajectory)
- [ ] Optimize that video's funnel: landing page + description links + end screen
- [ ] Produce companion blog post for top 5 videos (SEO compound effect)
- [ ] Target: 200+ subs/day on peak days, 5,000+ monthly views from SEO
