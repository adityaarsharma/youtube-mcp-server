# /youtube funnel

## The Plugin Company YouTube Funnel (Different From Ad-Revenue)

```
YouTube Video
    ↓ [Description link — widget page]
Widget/Feature Landing Page
    ↓ [Docs link — reduces friction]
Documentation
    ↓ [Buy button or free download CTA]
Free Download / Pro Purchase
    ↓ [Email sequence]
Active Plugin User
    ↓ [Renewal / Upsell to Agency plan]
Long-term Customer
```

**Research**: ConversionXL, 2024 — plugin demo videos convert at 2.8% avg, 6%+ for widget demos where the landing page has a live demo.

## Funnel Audit for Any Video

Check these elements in order:

### Layer 1 — YouTube Video
| Element | Present? | Fix If Missing |
|---------|----------|----------------|
| Widget/product link in first 5 lines of description | | Add immediately |
| Live demo link visible before "show more" | | Move to top of description |
| CTA mentioned at 25% of video | | Add in script/re-record or add cards |
| End screen linked to product page | | Add end screen card |

### Layer 2 — Landing Page
| Element | Present? | Fix If Missing |
|---------|----------|----------------|
| Live demo or video embed on page | | Add YouTube embed |
| Free download CTA visible above fold | | Move CTA up |
| Pricing link prominent | | Add pricing button |
| Docs link for support intent visitors | | Add docs link |

### Layer 3 — Conversion
| Element | Present? | Fix If Missing |
|---------|----------|----------------|
| YOUTUBE10 discount code works | | Test and fix |
| Free version available to reduce friction | | Ensure free tier is clear |
| 60-day money-back guarantee prominent | | Add trust signal |
